專為在家居士設計的身心觀照工具，將日常事務轉化為修行正念。

## 核心功能

### 財務記帳
收支平衡、功德基金分類、大數據統計、每月預算管理。

### 健康日誌
血壓血糖記錄、用藥追蹤、情緒與身體觀照。

### 孕母育兒
產檢紀錄、胎動追蹤、疫苗時程、親子共修。

### 毛孩手冊
寵物健康、餵食、醫療與陪伴記錄。

### 金剛藏密室
本地強加密私密筆記，重要事項安心存放。
